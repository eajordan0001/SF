# 超釩 Super Fan — 客戶查詢系統

## 📁 檔案結構

```
superfan/
├── index.html        主頁面
├── style.css         樣式（冰藍品牌配色）
├── app.js            程式邏輯（含試算表串接）
├── sw.js             Service Worker（離線快取）
├── manifest.json     PWA 設定（可安裝到手機桌面）
└── icons/
    ├── icon-192.png  App 圖示（192×192px）← 放入 SF logo
    └── icon-512.png  App 圖示（512×512px）← 放入 SF logo
```

## 🚀 部署步驟

### 1. Google 試算表設定

**欄位順序**（第 1 列為標題，從第 2 列起填入資料）：

| 欄 | 欄位名稱 | 必填 | 說明 |
|----|---------|------|------|
| A | 公司名稱 | ✅ | |
| B | 地址 | ✅ | 完整地址，App 會自動解析行政區 |
| C | 電話 | ✅ | |
| D | 聯絡人 | ✅ | |
| E | 注意事項 | 選填 | 顯示為黃色警示 |
| F | 標籤 | 選填 | 逗號分隔，可用：`今日`、`冷藏` |
| G | Maps連結 | 選填 | 見下方說明 |

**試算表共用設定**：
1. 右上角「共用」→「一般存取」→「知道連結的人」→「檢視者」

### 2. G 欄 Maps 連結填法（解決地址偏差問題）

1. 用手機或電腦開啟 Google Maps
2. 搜尋並確認找到**正確的地點**
3. 點「分享」→「複製連結」
4. 將連結貼入試算表 G 欄

**效果**：
- 有填 G 欄 → 詳細頁顯示「精準定位」綠色標籤，導航按鈕為藍色
- 沒填 G 欄 → 顯示「地址搜尋」橘色標籤，導航按鈕為橘色

> 💡 不需要一次全部填完，先處理常出問題的地址即可。

### 3. 放入 SF Logo

將超釩 LOGO 圖片分別存為：
- `icons/icon-192.png`（192×192 px）
- `icons/icon-512.png`（512×512 px）

### 4. 上傳至 GitHub Pages（免費）

1. 建立 [GitHub](https://github.com) 帳號
2. 新增 Repository（名稱任意，例：`sf-app`）
3. 上傳全部檔案（含 `icons/` 資料夾）
4. 進入 Settings → Pages → Source 選 `main` branch → Save
5. 等待約 1 分鐘，即可取得網址

**或使用 Netlify Drop（更簡單）**：
拖曳整個 `superfan` 資料夾到 https://app.netlify.com/drop

## 📱 司機安裝到手機

**Android（Chrome）**：  
開啟網址 → 右上角「⋮」→「新增到主畫面」→ 確認

**iOS（Safari）**：  
開啟網址 → 下方「□↑」分享鈕 → 「加入主畫面」→ 確認

安裝後使用體驗與原生 App 相同，且無需更新。

## 🏷️ 標籤說明

| 標籤值 | 顯示顏色 | 說明 |
|--------|---------|------|
| `今日` | 綠色 | 今日配送，可在「今日配送」篩選 |
| `冷藏` | 藍色 | 冷藏品，可在「冷藏品」篩選 |
| 有填注意事項 | 黃色警示 | 自動在「需注意」篩選顯示 |

多個標籤請用逗號分隔，例：`今日,冷藏`

## ❓ 常見問題

**Q：更改試算表後資料沒更新？**  
A：點 App 右上角重新整理按鈕，或等快取到期（預設 30 分鐘）。

**Q：沒有網路可以用嗎？**  
A：可以看到上次快取的資料，無法取得最新資料。

**Q：行政區篩選按鈕沒有出現？**  
A：請確認地址欄填入完整格式，例：`台南市東區東門路一段1號`，系統才能自動解析行政區。
